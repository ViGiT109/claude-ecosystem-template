## How to add a new ADR

1. Copy `_template.md` to `NNN-short-title.md` (sequential number).
2. Fill in all sections.
3. Add an entry to the index table above.
4. Commit: `docs(adr): add ADR-NNN short-title`.

## ADR statuses

- **Proposed** — under discussion, not yet decided
- **Accepted** — decision made, being implemented
- **Deprecated** — superseded by a newer decision
- **Superseded by ADR-NNN** — replaced
